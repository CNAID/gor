.. _HIDE:

====
HIDE
====
The **HIDE** command removes listed columns from the output of GOR or NOR queries. Note that columns 1 and 2 cannot be hidden when used with GOR queries.

Usage
=====

.. code-block:: gor

	gor ... | HIDE 3-5,7-

See also :ref:`TRYHIDE`.
